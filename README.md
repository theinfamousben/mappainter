# mappainter

Browser-based birdseye city map painter focused on road-first layout design.

## Getting Started

```bash
npm install
npm run dev
```

Open the local Vite URL shown in the terminal.

## Controls

- Select tool: click roads/buildings to edit or delete.
- Road tool: click to add points, double-click or press Enter to finish road.
- Building tool: drag to place a building footprint.
- Pan: hold Space and drag (or middle mouse drag).
- Zoom: mouse wheel.
- Cancel active drawing: Escape.
- Delete selected object: Delete/Backspace or panel button.
